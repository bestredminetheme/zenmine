<h2>Features: </h2>
* Customizable.
* Preset styles.
* Minimal. Clear.
* Hide sidebar. Fixed header.
* Hide/change footer text.
* Projects' tree display.
* Compatible with all browsers.
* Compatible Redmine 2.6 - 4.x
* Compatible with all RedmineUP plugins and many more.
